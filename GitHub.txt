COMMAND PROMPT CHANGE DRIVE

Type the drive letter followed by a colon (e.g., D:) and press Enter to switch to that drive. 
To change both the drive and the directory at the same time, use the command cd /d [drive letter]:\[folder].

USING GIT COMMIT COMMAND ON VSCODE TERMINAL

To put a commit message, press Insert + Enter (keys from your keyboard)
To exit from the commit message editor, Esc + Type ":wq"
